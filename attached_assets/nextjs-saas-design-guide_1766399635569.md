# Next.js SaaS Design Skill - Quick Reference

## What This Skill Does

Creates production-grade, modern Next.js SaaS applications and directories with distinctive designs that **don't look AI-generated**.

Specifically avoids:
- ❌ Generic purple gradients
- ❌ Overused fonts (Inter, Roboto, Space Grotesk)
- ❌ Cookie-cutter layouts and predictable sections
- ❌ Basic boxes with no personality

Instead provides:
- ✅ 5 distinctive aesthetic directions
- ✅ Comprehensive component patterns
- ✅ Purposeful animation strategies  
- ✅ Dashboard and data visualization patterns
- ✅ Next.js optimization best practices
- ✅ Replit deployment considerations

## How to Use This Skill

### 1. Install the Skill
Upload the `nextjs-saas-design.skill` file to Claude (if not already available)

### 2. Trigger the Skill
When building Next.js projects, the skill will automatically activate. You can also explicitly request:
- "Build me a SaaS dashboard using the nextjs-saas-design skill"
- "Create a landing page with distinctive design"
- "Help me design a directory site that doesn't look AI-generated"

### 3. Workflow
The skill will guide you through:
1. **Pre-Design Analysis** - Understanding product context and choosing aesthetic
2. **Typography & Colors** - Selecting distinctive fonts and intentional palettes
3. **Component Creation** - Building unique, non-generic components
4. **Animation** - Adding purposeful motion that enhances UX
5. **Optimization** - Implementing Next.js best practices

## Key Principles

1. **Choose ONE aesthetic direction** and commit fully
2. **Avoid generic patterns** at all costs
3. **Typography matters** - never use Inter/Roboto/Space Grotesk
4. **Intentional color choices** based on product context
5. **Break away from boxes** - use asymmetry and composition
6. **Purposeful animation** - guide attention, don't distract
7. **Next.js optimization** - SEO, images, fonts
